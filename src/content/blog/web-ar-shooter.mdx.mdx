# 概要
タイトル通り、ブラウザ上で動くARシューティングゲームをunityで作ったと言う話です。  
制作動機としては、大学で刺激を得た結果、久々にゲーム制作に復帰しました。  
そこで、久々にゲーム制作をするのであれば、何か初挑戦のジャンルが良いと考え、作りました。  
今のところコード的にもゲーム的にもかなり良い出来だと感じていますので、下のリンクから遊んで見てください。  
- [ゲームを遊ぶ](https://unityroom.com/games/gun_swap_shooter)
- [Githubリポジトリへ](https://github.com/Hosituka/GunSwapShooter)
# 技術スタック
- フレームワーク：Unity6
- Unityのライブラリ：UniTask
- シェーダー作成：ShaderGraph
- 3dモデル作成：Blender
- テクスチャ作成：Gimp
- コードレビュー要因：GeminiCLI※  

※GeminiCLIについて：コードは書かせていません、主にコードレビューやサイト構成を考えるのに使いました。
